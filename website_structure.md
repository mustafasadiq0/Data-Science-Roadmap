## 4. مسارات التعلم (Learning Paths)

### 4.1 مسار Excel و Google Sheets
- مقدمة عن Excel في تحليل البيانات
- الدوال الأساسية والمتقدمة
- إنشاء الرسوم البيانية
- جداول البيانات المحورية
- مشاريع عملية
- روابط لدروس Excel من مدونة مصطفى صادق

### 4.2 مسار Power BI و Data Studio
- مقدمة عن أدوات الذكاء التجاري
- إنشاء لوحات المعلومات التفاعلية
- ربط مصادر البيانات المختلفة
- تصميم التقارير الاحترافية
- مشاريع تطبيقية

### 4.3 مسار Python و Jupyter
- أساسيات البرمجة بـ Python
- مكتبات pandas و numpy
- تصور البيانات مع matplotlib و seaborn
- تحليل البيانات الإحصائي
- مشاريع تحليل البيانات الحقيقية
- روابط لسلسلة دروس Python من مدونة مصطفى صادق

### 4.4 مسار التصور التفاعلي (Data Visualization)
- مبادئ تصميم الرسوم البيانية
- أدوات التصور المختلفة
- إنشاء قصص البيانات
- أفضل الممارسات في عرض البيانات
